
# Matplotlip
This folder contains Jupyter Notebooks that introduce and explore Matplotlib, a powerful Python library used for data visualization. The notebooks cover both basic and advanced concepts, helping learners understand how to create a wide range of visualizations effectively.


## Folder Structure
```bash
Matplotlib_Practice/
├── README.md
├── .gitignore
├── Extra_Matplotlib.ipynb
├── Intro_Matplotlib.ipynb
├── example.png
```

## Learning Outcomes

By going through these notebooks, you’ll learn how to:

- Import and set up Matplotlib in Python.

- Create basic visualizations like line, bar, scatter, and pie charts.

- Customize plots with titles, labels, legends, and gridlines.

- Work with multiple plots using subplots.

- Control figure size, color, and style.

- Save visualizations as image files.



## Folder Contents

| File Name | Description |
|------------|-------------|
| **Intro_Matplotlib.ipynb** | Covers the fundamentals of Matplotlib — including importing libraries, basic plots (line, bar, scatter), customization of plots, labels, and legends. Ideal for beginners starting with data visualization in Python. |
| **Extra_Matplotlib.ipynb** | Extends the basics with advanced plotting features — subplots, styles, color maps, grid control, figure size adjustments, and saving figures. Useful for learners wanting to enhance their plotting skills. |


## Requirements

Make sure you have the following installed before running the notebooks:
```bash
pip install matplotlib numpy pandas
```

You can open the notebooks using Jupyter Notebook, JupyterLab, or VS Code with a Jupyter extension.

## Usage

1) Clone or download this repository.

2) Open the notebooks in Jupyter or VS Code.

3) Run the cells sequentially to visualize outputs and learn step by step.
